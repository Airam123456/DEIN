VENTANA PRINCIPAL

Al ejecutar la aplicación se abrirá una ventana principal desde la cual tendremos varias funciones disponibles. A primera vista se puede distinguir una parte izquierda para gestionar alumnos, la parte derecha para gestionar los libros, un botón abajo para realizar prestamos de libros y una barra superior con varias opciones para gestionar prestamos, un apartado de informes y una pestaña de ayuda.

- El apartado para alumnos: Aquí se podrán visualizar todos los alumnos disponibles, además de tener la opción de crear nuevos alumnos, modificarlos o borrar los ya existentes. Para ello, puedes pulsar los botones a la derecha de la palabra Alumnos o haciendo click derecho encima de la tabla de libros. A continuación solo tendrás que introducir los datos correspondientes.

- El apartado para libros: Aquí se podrán visualizar todos los libros disponibles, además de tener la opción de crear nuevos libros, o modificar o borrar los ya existentes. Para ello, puedes pulsar los botones a la derecha de la palabra Libros o haciendo click derecho encima de la tabla de libros. A continuación solo tendrás que introducir los datos correspondientes.

- Realizar préstamo: Para realizar un préstamo simplemente tenemos que pulsar el botón, el cual hará aparecer una ventana desde donde podremos gestionar el préstamo que queremos hacer. Si el prestamos se realiza correctamente aparecerá un pdf con la información del préstamo así como fecha de devolución.

- Barra superior: En este menú hay 4 opciones diferentes: préstamos (abre una ventana para gestionar los préstamos en curso), histórico de préstamos (abre una ventana para ver todos los préstamos que se han hecho), informes (ofrece la opción de visualizar varios informes) y ayuda (ofrece una guía rápida de la aplicación).
